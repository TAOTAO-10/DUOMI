# 多米（Domi）宠物介绍网站

这是一个完整的静态宠物介绍网站项目，使用中文界面，包含首页、关于多米、成长记录、日常照片墙和多米档案页面。

当前页面已经替换为多米的真实照片，照片资源位于 `assets/photos/`。

## 项目结构

```text
.
├── index.html
├── about.html
├── timeline.html
├── gallery.html
├── profile.html
├── css/
│   └── styles.css
├── js/
│   └── main.js
└── assets/
    └── photos/
        ├── domi-crown.jpg
        ├── domi-curious.jpg
        ├── domi-explorer.jpg
        ├── domi-goodluck.jpg
        ├── domi-mealtime.jpg
        ├── domi-nap.jpg
        ├── domi-outfit.jpg
        ├── domi-peek.jpg
        ├── domi-sofa.jpg
        └── domi-toy.jpg
```

## 使用方式

直接双击打开 `index.html` 即可浏览网站。照片已经做过网页尺寸压缩，后续如果要替换照片，把新图片放入 `assets/photos` 并修改页面中的图片路径即可。
